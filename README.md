# diabetes-ML-project

Machine Learning project for diabetes prediction using multiple classification models.

## Models Used
- Logistic Regression
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

## Preprocessing Steps
The dataset was preprocessed before training, including:
- Handling missing values
- Feature scaling
- Data cleaning
- Train/Test split

## Technologies
- Python
- Jupyter Notebook
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn

## Project Goal
Predict whether a patient has diabetes based on medical attributes.

## Dataset
The project uses a diabetes dataset for binary classification tasks.
